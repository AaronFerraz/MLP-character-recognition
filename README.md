### Multilayer Perceptron
Este trabalho foca na implementação de uma rede neural artificial Multilayer Perceptron (MLP) com uma camada escondida. A principal diretriz é que a rede deve ser implementada sem o uso de bibliotecas especializadas em redes neurais artificiais, sendo treinada com o algoritmo Backpropagation em sua versão de Gradiente Descendente. O desenvolvimento abrange a modelagem, implementação, testes e análise de resultados , utilizando o conjunto de dados CARACTERES COMPLETO para as etapas principais. As atividades incluem a execução do treinamento (com e sem validação cruzada ou parada antecipada) , um estudo de parâmetros (grid search) e o teste da rede para avaliar seu funcionamento.
